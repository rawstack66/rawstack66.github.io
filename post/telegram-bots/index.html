Telegram Bots推荐

2020-04-18 / 9 min read

Telegram 在 2015 年 6 月发布了机器人平台，经过几年的发展，基于其丰富的自定义接口和开放的特性，把Telegram这样一款IM工具也扩化成了瑞士军刀，也已然融入到了个人和社区用户的日常Telegram使用中。这里推荐一些TG机器人，帮助大家提高Telegram的使用体验。

Bots 商店

在 Raw栈导航 最下面的 Telegram频道 - Bot商店，你可以找到很多发布和更新Bots的频道或机器人（非套娃），跳转查看就可以获取各种实用 Bots。

官方 Bots

  • @BotFather BotFather，可以创建和管理机器人，Bot 教父
  • @BotSupport Bot Support，开发中遇到的问题和建议可以给这个机器人发送反馈
  • @SpamBot Spam Info Bot，+86手机用户注册后需要到这里进行对话解封操作，也可以查询自己有没有在官方的账号限制名单中
  • @telegraph Telegraph，登录和管理 Telegraph，获取文章的阅读统计数据
  • @Stickers Telegram贴图和贴图包的建立、修改、统计与移除都是靠「 Stickers」这个官方的机器人运作
  • @previews Previews，反馈和评论帮助官方完善 Web 文章的 Instant View（即时预览）功能
  • @GmailBot Gmail Bot，在Telegram内收发gmail
  • @GameBot GameBot，Telegram游戏平台的demo机器人，在Telegram，你甚至可以打游戏
  • @IFTTT IFTTT，连接你的群组或频道到360多种服务

服务

系统功能

  • @getidsbot GetIDs Bot，此机器人可以查看 Telegram 帐号的真实ID、注册日期等信息
  • @RawDataBot Telegram Bot Raw，获取频道、群组和用户信息
  • @getuseridbot 发送信息即刻获取用户ID
  • @LivegramBot 可以搭建自己的私聊机器人，搭配 Botfather 使用
  • @Manybot 创建自己的机器人，功能很多
  • @CorsaBot 发送网页转发成 Instant View 文章
  • Telegram-Anti-Revoke Telegram 防撤回插件
  • @IVrobot Instant View Robot，快速创建Telegraph

文件

  • @tlgurbot tlgur，发送图片、视频等各种文件获取公共链接，便于引用、分享和存档，速度很快，Telegram上的文件托管服务
  • @TGhost_bot TGhost — file sharing，发送文件获取下载链接
  • @GetPublicLinkBot 嫌电报直接下载大文件速度慢，可以直接用这个机器人转存到自己的谷歌网盘或者获取外部下载链接满速下载
  • @uploadbot URL Uploader，发送文件url即可下载文件到 Telegram，如 Google Drive、Instagram分享链接，甚至还可以下载磁力
  • @DrWebBot Dr.Web，发送任何文件查看有无后门或病毒
  • @Gdriveit_bot Google Drive Bot，支持上传和管理 Google Drive

工具

  • @AlertBot Alert Bot，可以在指定时间后发送指定信息给自己，一个提醒工具
  • @fanhuaji_bot 繁化姬器人，此机器人可以将文字或句子繁简转换、台湾化、中国化、香港化、拼音化、火星化等，非常实用！
  • @sticker_time_bot Sticker Time Bot，整点报时机器人
  • @QRCodeRoBot QR Code Bot，识别二维码
  • @WooMaiBot 干物妹！小霾，多功能机器人
  • @SovietJokesBot 苏联笑话生成器
  • @LinkGeneratorBot URL Shortener 🔗，短网址生成，前缀为 bit.ly、q.gs、tinyurl.com
  • @referbot Shorten URL | ume.la，ume.la短网址，可查看链接打开统计数据
  • Telegram SMS 一个在Android手机上跑的Telegram Bot，能帮你转发短信，监听未接通来电和监控电池电量；这一切都是直接调用Telegram Bot API的，不需要经过任何第三方服务器

邮箱

  • @etlgr_bot Telegram Email，临时邮箱服务，@etlgr.com 后缀
  • @fakemailbot Fake Mail，临时邮箱服务，@hi2.in 后缀

订阅 / RSS

  • @TheFeedReaderBot Feed Reader Bot，除了RSS功能，还可以订阅YouTube、Twitter、Instagram、Netflix等媒体内容
  • @RSSFlowBot flowerss，一个支持应用内阅读的RSS机器人
  • @RustRssBot 中文 Telegram RSS 机器人
  • @pstrbot PosterBot，自动抓取 Instagram, Twitter, VK, Tumblr, You Tube 和 RSS 帖子/更新到频道
  • @Channel2RSSBot channel2rssBot，将频道内容输出成 RSS 和 ATOM 订阅

App

  • @PTRankBot Beemo，可以帮助你搜索和分享 iOS/MacOS 应用
  • @apkdl_bot APK Downloader，从 apkdl.in 下载安卓 App

英语

  • @LexicoBot 调用 Wordnik 的多功能词典 Bot，可查释义、同反义词、例子等等

媒体

  • @getmediabot GetMediaBot，下载图片、歌曲、视频及更多

音乐

  • @Deezer_spotify_bot 下载 Deezer 和 Spotify上的歌曲，可以搜索歌手、专辑、单曲进行下载，带轨道、发行时间、海报等歌曲信息，320kbps（可选下载音质），非常好用
  • @MusicHuntersBot Music Hunters，输入 Deezer 和 Spotify 的单曲链接下载，带歌曲信息
  • @YTAudioBot YouTube Audio Bot，下载YouTube视频的音频，支持20多种语言，发送视频链接即可下载
  • @vkmusic_bot VK Music Bot，曲库超大，有几十万用户，还可以搜中文歌，Spotify、SoundCloud也不在话下
  • @scloud_bot 下载 SoundCloud里的歌曲，可直接搜索、发送链接下载
  • @scdlbot Music Downloader，Download MP3s of tracks/sets from SoundCloud, Bandcamp, YouTube with tags.
  • @spotifybot 搜索 Spotify 数据库并下载或分享歌曲的预览
  • @ChinoNyanBot チノ喵，网易云搜索下载机器人
  • @iLyricsBot 歌词搜索机器人，歌名加一点歌词内容即可精准返回歌词，非常实用
  • @mp3toolsbot MP3 Tools，可进行转换、剪辑、编辑歌曲信息等操作
  • @MuzzzBot 🎶 MuzzzBot 🎶，曲库强大，还可以通过麦克风识别歌曲

视频

  • @utubebot utubebot，支持YouTube、Dailymotion、Vimeo等多个视频平台，可以分别按照画质和音质下载视频、音频文件，还可以播放热门视频、订阅油管频道，功能过于强大，建议Telegrammer人手一个

图片与贴纸

贴纸生成

  • @big_text_bot Text sticker，此机器人可以建立大字图，只要输入「@big_text_bot +指定文字」就可以生成大字图

  • @wangjingze_bot 王境泽，快速创建王境泽表情包

编辑与管理

  • @GifReverseBot GIF Reverse Bot，GIF / short video倒放
  • @FavoriteStickersBot Favorite Stickers，创建自己的贴纸收藏夹，也可以创建贴纸包
  • @giftubebot GifTube，截取 Youtube 或 Vimeo 片段然后生成 Gif

下载贴纸

  • @stickerset2packbot StickerSetBot，此机器人可以打包贴纸并下载，也可以打包贴纸成贴纸包
  • @Stickerdownloadbot Sticker Downloader，下载贴纸和贴纸包
  • @GIFDownloader_bot TG Downloader，下载 GIF，静态贴纸和静态贴纸集
  • @dlgifbot DownloadGifBot，下载GIF或贴纸

图片

  • @MaribelBot Maribel，此机器人可以获取网络上的ACG插图，来源相当多元
  • @Instasave_bot InstaSave，发送帖子link下载Instagram图片，发送@用户名下载头像和简介
  • 输入框内打出 @bing / @pic / @gif 可进行 Bing / Yandex / Giphy 图片搜索，贴纸库不够用的时候可以派上用场
  • @reverseSearchBot Reverse Search B⌕t，反向搜索机器人，发送图片/GIF/贴纸即可，或者发送里番截图返回了节目信息以及时间戳
